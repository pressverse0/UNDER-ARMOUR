# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-03-01

### Added
- Initial release of Under Armour E-Commerce Platform
- Product catalog with categories (Men, Women, Shoes, Sports)
- Shopping cart with add/remove/update functionality
- Wishlist feature for saving favorite products
- Stripe checkout integration for secure payments
- User account system with profile management
- Order history and tracking
- Product detail pages with image galleries
- Review and rating system
- Social media sharing functionality
- Custom toast notification system
- Responsive design for all devices
- Dark/light theme support
- 404 error page with animations
- Support pages (FAQ, Contact, Returns, Shipping, Size Guide)
- Footer with company information and links
- Header with navigation and cart/wishlist icons

### Features
- **Product Browsing**
  - Grid layout with product cards
  - Category filtering
  - Product images with hover effects
  - Price display with sale badges
  - Quick actions (add to cart, wishlist)

- **Shopping Cart**
  - Slide-out sidebar
  - Real-time price calculations
  - Quantity management
  - Persistent state
  - Toast notifications

- **Checkout**
  - Customer information form
  - Shipping address
  - Stripe payment integration
  - Order confirmation page
  - Success/cancel handling

- **User Account**
  - Profile management with edit modals
  - Order history with status tracking
  - Order tracking modal with timeline
  - Wishlist management
  - Account settings (security, notifications, payment, language)
  - Logout functionality

- **Product Details**
  - Image gallery with thumbnails
  - Color and size selection
  - Quantity selector
  - Add to cart/wishlist
  - Share functionality (Facebook, Twitter, LinkedIn, Email, Copy Link)
  - Review submission modal
  - Tabbed content (Description, Features, Reviews)
  - Shipping information

- **UI/UX**
  - Bold Under Armour branding
  - Sketchy borders and shadows
  - Custom animations
  - Toast notifications for all actions
  - Loading states
  - Error handling

### Technical
- Next.js 15.2.6 with App Router
- TypeScript 5
- Tailwind CSS 3.4
- Radix UI components
- shadcn/ui component library
- Stripe payment processing
- React Hook Form with Zod validation
- Lucide React icons
- Vercel deployment

### Documentation
- Comprehensive README.md
- Feature documentation
- Technology stack documentation
- Project structure guide
- Styling guide
- Contributing guidelines
- Code of Conduct
- Security policy
- Use cases documentation
